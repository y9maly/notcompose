
Я хочу, чтобы в самом runtime и Composer не было встроено то, что ноды могут быть "грязными" и их надо рекомпозировать.

Этот плагин просто добавляет DirtyComposition extension к нодам, что позволяет пометить их композицию как "грязную".

И другой плагин, например который отслеживает изменение State, использует этот маркер чтобы пометить композицию как грязную.
