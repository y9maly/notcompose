
Объявляет RecomposeLambda extension.
Если нода имеет этот extension, то она поддерживает рекомпозицию самой себя.

То-есть, если только эту ноду нужно рекомпозировать, а рекомпозировать родителя не обязательно (его стейт не изменился),
то это можно сделать вызвав RecomposeLambda лямбду.

see also: [function notcompose-terminal/Layout]
